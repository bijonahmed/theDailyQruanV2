# TheDailyQruanV2
Learn the Quran, One Day at a Time
The Daily Quran

The Daily Quran is a web application that helps users read, listen, and understand the Quran daily. It provides:

Full Quran text in Arabic with proper tajweed

Translations in multiple languages (English, Bangla, Urdu, Hindi)

Audio recitation for each Surah and Ayah

Daily Juz reading for structured learning

Search functionality to find Surahs by name or number

Responsive design for mobile and desktop

Perfect for anyone looking to learn, reflect, and connect with the Quran in a structured and easy way.

Tech Stack: React.js, React Router, Bootstrap 5, REST API (alquran.cloud), CSS
